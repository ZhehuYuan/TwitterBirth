1. You need to install python3.5 and numpy and sklearn packet to run this program.

2. You must make sure the data file "Females", "Males", and "us" are in the same folder with the .py file. 

3. This code can be used to train either screen name, name, or bios. So, before run the program, 
you should follow the common marked with **** to make some change to the code base on
what data you want to train. (Don't forget to undo these change and redo step3 before training 
next data).

4. In command line, tap following to run:
python3 Unsupervise_Learning.py

5. Also, you can use pycharm to run this ".py" file with python3.5 interupter and packet installed.
